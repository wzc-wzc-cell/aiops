# 基于机器学习的智能日志异常检测与告警系统
项目背景：
在传统运维中，日志规模大、格式复杂，依赖人工排查或静态规则进行异常发现，存在响应滞后、误报率高、难以发现未知异常等问题。
本项目通过引入 无监督机器学习算法，对系统日志行为进行建模，实现日志异常的自动识别、告警与可视化分析，提高系统稳定性和运维效率。


# 项目职责
	•	负责 服务器与应用日志的采集、清洗与结构化处理
	•	设计日志 特征工程方案，提取系统运行关键指标
	•	基于 Isolation Forest 等无监督模型 实现日志异常检测
	•	实现异常日志 自动告警与结果可视化展示
	•	对模型效果进行分析与优化，降低误报率


# 技术架构
	•	日志采集：Filebeat / Logstash
	•	日志存储与检索：Elasticsearch
	•	数据处理：Python、Pandas
	•	异常检测：Isolation Forest（PyOD / scikit-learn）
	•	告警通知：邮件 / 钉钉 / 企业微信 Webhook
	•	可视化：Kibana / Grafana
